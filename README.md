> This is a Danger Plugin that ensures nice and tidy commit messages.

## Usage

Install:

```sh
yarn add danger-plugin-commit-lint --dev
```

At a glance:

```js
// dangerfile.js
import { commitLint } from 'danger-plugin-commit-lint'

commitLint()
```
